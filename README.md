# goinit    


goinit  create  a   go  project  in  $GOPATH/src/    


## Start using it

Download and install it:    
`$ go get -u -v github.com/alphayan/goinit`


## Project directory hierarchy
``` 
<project>/       
    |- -main.go     
    |- -config.go       
    |- -db.go       
    |- -redis.go        
    |- -rabbitmq.go     
    |- -router.go       
```
